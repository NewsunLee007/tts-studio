Web TTS 制作台（英语听力音频）

架构草图：浏览器管理片段与风格；服务端代理调用 TTS 大模型并合成 MP3；最终可下载整段音频。

核心交互

  • • 文本片段列表：增删改、排序、试听
  • • 片段类型：TTS / 无声（n 秒）
  • • 导出：合成为 MP3 并下载

风格与音色

  • • 内置常用风格：播音员 / 对话 / 讲解
  • • 每段可选 voice/style 或按全局默认
  • • 默认面向英语（en-US / en-GB）

安全与成本

  • • API Key 不落库（仅浏览器保存）
  • • 服务端只做代理与合成，不记录明文 key
  • • 可选：按 IP / 项目限流，防止滥用
